# Intelligent Code Search Engine

A mini Google-like search engine for programming code snippets.

## Features
- 🔍 Search code snippets by keywords
- 💾 Store full programming code
- ⚡ Fast retrieval using MongoDB
- 🎨 Clean Google-like UI
- 💻 Syntax highlighted code display

## Tech Stack
- React
- Node.js
- MongoDB

## How to Run
1. Start backend: node server.js
2. Start frontend: npm start
